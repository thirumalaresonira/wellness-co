# VedaOS - AI-Powered Wellness ERP

VedaOS is a premium, state-of-the-art ERP system designed for luxury wellness centers and spas.

## Tech Stack
- **Frontend**: React, Vite
- **Styling**: Vanilla CSS (Tailwind patterns), Framer Motion
- **Icons**: Lucide React
- **Charts**: Recharts

## UI/UX Design
- **Theme**: Pastel luxury spa aesthetic (Blush Pink, Mint Green, Lavender).
- **Aesthetics**: Glassmorphism, rounded corners, soft shadows, smooth transitions.
- **AI-First**: Dedicated modules for AI Recommendations and AI Voice Follow-ups.

## Key Screens
- **Landing Page**: http://localhost:5173/
- **ERP Dashboard**: http://localhost:5173/dashboard

## Modules Implemented
- CRM (Client Profiles)
- Smart Scheduling
- Therapy Session Tracking (with Video Analysis placeholder)
- Billing & Payments
- AI Follow-up Calling System
- AI Recommendation Engine
- Multi-branch Analytics
- Real-time Performance Dashboards
